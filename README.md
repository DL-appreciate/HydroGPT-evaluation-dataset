# HydroGPT-evaluation-dataset
A large language model capability evaluation set containing 200 Q&amp;A pairs in the field of hydraulic structures.

This is a large language model capability evaluation set containing 200 questions in the field of hydraulic structures. The evaluation set comprehensively covers major sub-disciplines of hydraulic engineering including hydraulic tunnels, dams, hydropower stations, sluice gates, and flood discharge facilities in terms of content, and evenly includes four question types: multiple-choice, fill-in-the-blank, true/false, and short answer.
